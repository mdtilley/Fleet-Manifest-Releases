Fleet Manifest

Compatible with macOS 14+/iOS/iPadOS 17+

Intel/Apple Silicon Compatible

I got tired of having so many devices either at home or at work that I could barely keep track of, so I vibecoded an app to keep track of everything


FEATURES

* Device Inventory List - Log, track, and manage all electronics in a centralized list.
* Custom Categories - Split your electronics into their own categories, complete with colored tabs that are user set.
* Cross Platform Support - Runs on macOS, but can be compiled natively for iOS and iPadOS. JSON exporting and importing allows you to transfer inventories between apps.
* Scanner Integration for iOS/iPadOS - Allows the camera to be used to scan in Serial Numbers.
* Global Parts - Add your loose parts to this section, complete with cost and serial numbers/batch numbers. Sunk Cost/Parts Installed allows you to view how much in parts has been used per-device.


* Customization - Themes change the background of the main menu.


Installation:

macOS:
Head to Releases and download the DMG. Move the App to your /Applications folder. Your security flag will raise, so go to Privacy and Security in your settings to open it for the first time.

iOS/iPadOS:
Clone this git repo, and then open the folder in Xcode. At the top, select your iDevice (must have Developer Mode enabled, and the device has to be paired to your computer). Press the play button, and it will compile and run on your iDevice.
